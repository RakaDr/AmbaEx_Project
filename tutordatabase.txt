Langkah 1: Login
Buka aplikasi SQL Shell (psql).

Tekan Enter untuk Server [localhost].

Tekan Enter untuk Database [postgres].

Tekan Enter untuk Port [5432].

Tekan Enter untuk Username [postgres].

Saat muncul Password for user postgres:, ketik password kamu (misal: raka).

Catatan: Saat mengetik password, layar tidak akan menampilkan bintang atau huruf. Ketik saja lalu tekan Enter.

Langkah 2: Buat & Masuk Database
Ketik perintah ini lalu Enter:

SQL

CREATE DATABASE db_ambaex;
Ketik perintah ini untuk masuk ke database baru:

SQL

\c db_ambaex
(Harus muncul tulisan: "You are now connected to database...")

Langkah 3: Jalankan Kode SQL
Copy kode SQL lengkap di bawah ini.

Kembali ke layar hitam SQL Shell.

Klik Kanan mouse di layar hitam (ini otomatis melakukan Paste).

Tunggu proses berjalan.

Jika berhenti tapi belum selesai, tekan Enter sekali.

-- ==========================================
-- DATABASE SCRIPT: AMBA EX PROJECT
-- ==========================================

-- 1. BERSIHKAN TABEL LAMA (RESET)
-- Menggunakan CASCADE agar tabel yang berelasi ikut terhapus bersih
DROP TABLE IF EXISTS feedback CASCADE;
DROP TABLE IF EXISTS tracking CASCADE;
DROP TABLE IF EXISTS paket CASCADE;
DROP TABLE IF EXISTS layanan CASCADE;
DROP TABLE IF EXISTS admin CASCADE;

-- ==========================================
-- 2. MEMBUAT TABEL BARU
-- ==========================================

-- Tabel ADMIN (Untuk Login Staff)
CREATE TABLE admin (
    id SERIAL PRIMARY KEY,
    username VARCHAR(50) UNIQUE NOT NULL,
    password VARCHAR(255) NOT NULL,
    nama_lengkap VARCHAR(100)
);

-- Tabel LAYANAN (Untuk Data Harga & Dropdown)
CREATE TABLE layanan (
    id SERIAL PRIMARY KEY,
    nama_layanan VARCHAR(50),
    estimasi VARCHAR(50),
    harga_per_kg DECIMAL(10,2)
);

-- Tabel PAKET (Data Utama Transaksi)
CREATE TABLE paket (
    no_resi VARCHAR(20) PRIMARY KEY,
    tanggal_input TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    nama_pengirim VARCHAR(100),
    telp_pengirim VARCHAR(20),
    kota_asal VARCHAR(50),
    nama_penerima VARCHAR(100),
    telp_penerima VARCHAR(20),
    alamat_penerima TEXT,
    kota_tujuan VARCHAR(50),
    berat DECIMAL(5,2),
    layanan VARCHAR(50),
    isi_paket VARCHAR(100),
    biaya DECIMAL(15,2),
    status_terakhir VARCHAR(50) DEFAULT 'PICK UP'
);

-- Tabel TRACKING (Riwayat Perjalanan Paket - Relasi ke Paket)
CREATE TABLE tracking (
    id SERIAL PRIMARY KEY,
    no_resi VARCHAR(20) REFERENCES paket(no_resi) ON DELETE CASCADE,
    waktu TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    lokasi VARCHAR(100),
    status VARCHAR(50),
    keterangan TEXT
);

-- Tabel FEEDBACK (Pesan Masuk dari User)
CREATE TABLE feedback (
    id SERIAL PRIMARY KEY,
    nama_pengirim VARCHAR(100),
    email VARCHAR(100),
    subjek VARCHAR(100),
    pesan TEXT,
    tanggal_kirim TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
